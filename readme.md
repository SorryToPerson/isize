## isize

一个通用的图标处理网站

#### 功能介绍：

1. 支持用户上传图片，支持用户裁剪自己的图片（正方形）
2. 根据用户上传的图片，针对不同的平台：WEB、APP、小程序等，生成对应尺寸的图标（从1024*1024到16*16）
3. 支持一键导出不同平台的图片，并且支持压缩导出，单个导出
4. 其中App平台中，要注意不同平台的App图标规范，比如IOS、macOS不同平台的图标设计规范，安卓APP、windowsAPP 等
5. 其中，图片处理部分，如果前端WEB/APP 客户端能够自行处理那就自行处理，处理不了再交给后端处理并返回
6. 所有存储/历史记录使用客户端本地，线上不做任何存储

### 技术栈：

web：React
App：React Native（兼容安卓、苹果、华为）
后端：nestjs
